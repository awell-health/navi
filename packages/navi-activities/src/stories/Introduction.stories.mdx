# Navi Activities

Welcome to the **Navi Activities** component library! This package provides production-ready React components for healthcare applications with a unified event system inspired by Stripe Elements.

## üèóÔ∏è Architecture

All activity components share the same external API for consistency:

```tsx
interface ActivityProps<T> {
  data: T;                    // Activity-specific data
  onReady?: (event) => void;  // Component ready
  onComplete?: (event) => void; // Activity complete
  onFocus?: (event) => void;  // Field focused
  onBlur?: (event) => void;   // Field blurred
}
```

## üìö Component Categories

### Activity Components
- **FormActivity** - Interactive forms with multiple question types
- **MessageActivity** - Rich content display (HTML/Markdown)
